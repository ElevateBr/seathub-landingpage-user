# SeatHub Landing Page

Landing page profissional para conversão de clientes interessados em espaços de coworking.

## 📁 Estrutura de Arquivos

```
seathub-landingpage-user/
├── index.html          # Página principal HTML
├── styles.css          # Estilos CSS organizados
├── script.js           # JavaScript principal (funcionalidades)
├── translations.js     # Sistema de tradução (PT/EN/ES)
└── README.md           # Documentação do projeto
```

## 🎯 Funcionalidades

### ✨ Design e UX
- **Layout Responsivo**: Mobile-first design
- **Paleta de Cores**: Amarelo (#ffd700), Preto (#000000), Branco (#ffffff)
- **Imagens de Background**: Unsplash com efeito parallax
- **Animações Suaves**: Hover effects e transições

### 🌐 Sistema de Tradução
- **3 Idiomas**: Português (PT), Inglês (EN), Espanhol (ES)
- **Tradução Dinâmica**: JavaScript com sistema completo
- **Seletores de Idioma**: Botões interativos no header
